# Desktop App (Future)

Tauri-based desktop wrapper for DysLex AI.

## Planned Features

- Native system tray integration
- Offline mode with local ONNX model
- System-wide text correction
- Keyboard shortcuts
- Cross-platform (Windows, macOS, Linux)

## Tech Stack

- Tauri (Rust)
- Frontend: Shared React codebase from `../frontend`
