# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial project structure
- React + TipTap frontend with four writing modes
- FastAPI backend with two-tier LLM processing
- PostgreSQL database schema for error profiles
- Docker configuration for full stack deployment
- Claude Code integration with hooks and agents
- Dyslexia-friendly UI with Cream and Night themes
- Voice input/output support via NVIDIA NIM
- Passive learning system for adaptive corrections

### Security
- Input validation on all API endpoints
- SQL injection prevention via SQLAlchemy ORM
- XSS prevention in React frontend
